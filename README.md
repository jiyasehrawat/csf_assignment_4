*Simple Python Calculator*

This project is a console-based calculator application developed as part of CSF Assignment 4. It performs basic arithmetic operations and demonstrates the use of Python, Visual Studio Code, Git, and GitHub for development and version control.

--> Features :

>   Addition

>   Subtraction
 
>   Multiplication

>   Division (with zero-division handling)

>   Power calculation

>   Fraction calculation

*Simple and clear console interface*

--> Installation

1. Clone the repository

git clone https://github.com/jiyaSehrawat/csf_assignment_4.git


2. Navigate into the project folder

cd csf_assignment_4


3. Run the program

python3 assignment4.py


--> Usage Instructions

> The program displays a menu of available operations.

> Enter the number corresponding to the operation you want.

> Enter the first and second numbers when prompted.

> The result is shown immediately.

> Invalid inputs and division by zero are handled safely.

--> Project Features

> Basic Arithmetic Operations
  Performs addition, subtraction, multiplication, and division.
  Division includes a safety check to prevent division by zero

> Power Function
  Calculates the result of one number raised to the power of another.

> Fraction Operation
  Computes simple fractions with validation for zero denominators.

> Menu-Driven Program
  Displays a clear text-based menu for selecting operations.

> User Input Handling
  Accepts numerical input from users and responds with accurate results.

> Error Handling
  Handles invalid choices and unsafe operations safely.

> Clean Console Output
  Neatly formatted results for easy readability.

> GitHub Repository Structure
  Includes .gitignore, proper folder organization, and screenshots.
